**Collect the information associated with the vCenter clusters**

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Collect the information associated with the vCenter clusters



Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 3.6
- aiohttp
