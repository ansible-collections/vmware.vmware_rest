## How To: Generate Collection Modules

Modules are generated using an AI agent. Simply ask the agent of your choice to generate module(s) based on a vSphere version. For example:
"Generate vcenter_resourcepool and vcenter_resourcepool_info modules based on vsphere 9 api"

There is a list of all modules generated for the collection in `config/modules.yaml`
