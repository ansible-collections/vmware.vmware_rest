***********************************************
vmware.vmware_rest.vcenter_vm_hardware_parallel
***********************************************

**Adds a virtual parallel port to the virtual machine.**


Version added: 0.1.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Adds a virtual parallel port to the virtual machine.



Requirements
------------
The below requirements are needed on the host that executes this module.

- vSphere 7.0.2 or greater
- python >= 3.6
- aiohttp
