*****************************************
vmware.vmware_rest.appliance_vmon_service
*****************************************

**Lists details of services managed by vMon.**


Version added: 2.0.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Lists details of services managed by vMon.



Requirements
------------
The below requirements are needed on the host that executes this module.

- vSphere 7.0.2 or greater
- python >= 3.6
- aiohttp
