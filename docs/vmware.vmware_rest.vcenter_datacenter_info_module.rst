******************************************
vmware.vmware_rest.vcenter_datacenter_info
******************************************

**Collect the information associated with the vCenter datacenters**


Version added: 1.0.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Collect the information associated with the vCenter datacenters



Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 3.6
- aiohttp
