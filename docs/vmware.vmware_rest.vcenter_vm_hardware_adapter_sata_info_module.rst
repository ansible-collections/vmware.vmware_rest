********************************************************
vmware.vmware_rest.vcenter_vm_hardware_adapter_sata_info
********************************************************

**Returns information about a virtual SATA adapter.**


Version added: 0.1.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Returns information about a virtual SATA adapter.



Requirements
------------
The below requirements are needed on the host that executes this module.

- vSphere 7.0.2 or greater
- python >= 3.6
- aiohttp
