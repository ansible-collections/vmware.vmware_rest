************************************
vmware.vmware_rest.vcenter_host_info
************************************

**Collect the information associated with the vCenter hosts**


Version added: 1.0.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Collect the information associated with the vCenter hosts



Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 3.6
- aiohttp
